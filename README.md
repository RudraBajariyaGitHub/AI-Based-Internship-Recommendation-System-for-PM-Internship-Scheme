# AI-Based-Internship-Recommendation-System-for-PM-Internship-Scheme
Multi-stage ML system for personalized, fair, explainable internship recommendations under India’s PM Internship Scheme. Uses Sentence-Transformers + FAISS retrieval, ensemble ranking, and fairness re-ranking. Outputs skills gap, SHAP insights, confidence, and upskilling advice via a FastAPI backend.
